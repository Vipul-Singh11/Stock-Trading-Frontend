# 📊 Stock Trading Frontend

Frontend application for the Stock Trading Simulation Platform built with React and Vite.

This dashboard provides a modern trading experience with portfolio management, stock market visualization, order placement, trade history tracking, and order book monitoring while integrating with the backend microservices ecosystem.

---

## 🚀 Features

### 🔐 Authentication

* JWT-based Login & Registration
* Protected Routes
* Session Management

### 📈 Dashboard

* Wallet Balance Overview
* Reserved Balance Tracking
* Available Balance Monitoring
* Portfolio Summary
* Recent Trade Activity

### 📊 Stocks

* Live Stock Listings
* Company Information
* Current Market Prices
* Interactive Stock Charts
* Search & Filtering

### 💼 Portfolio

* Holdings Overview
* Available Shares
* Reserved Shares
* Portfolio Allocation Visualization
* Portfolio Distribution Analytics

### 📝 Orders

* Place BUY Orders
* Place SELL Orders
* Cancel Pending Orders
* Track Order Status
* Partial Fill Support

### 📖 Trade History

* Executed Trade Records
* Search & Filtering
* Sortable Trade Table

### 📚 Order Book

* Buy Order Visualization
* Sell Order Visualization
* Best Bid / Best Ask Display
* Auto Refresh Support

---

## 🏗️ Backend Services

The frontend integrates with the following microservices:

| Service                 | Port |
| ----------------------- | ---- |
| User Service            | 8081 |
| Stock Price Service     | 8082 |
| Order Service           | 8083 |
| Matching Engine Service | 8084 |
| Portfolio Service       | 8085 |

## 🚀 Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

---

## 📌 Note

This repository contains only the frontend application.

The backend is implemented as independent Spring Boot microservices and maintained in separate repositories.
